## 年级自动换算（与英语专名词表共用）

**学年切换日**：每年 **9 月 1 日**起算新学年。

给定**入库日期**日期 `d`：

1. 若 `d` 的月份 **≥ 9**，学年起始年 `Y = d` 的年份；否则 `Y = d` 的年份 **− 1**。
2. 查下表得到**年级**：

| 学年起始年 Y（当年 9 月起） | 年级 |
|:---:|:---|
| ≤2024 | 小学四年级及以下（按 Y 递减：2024→小四，2023→小三…） |
| **2025** | **小学五年级** |
| **2026** | **预备班** |
| **2027** | **初一** |
| **2028** | **初二** |
| **2029** | **初三** |
| **2030** | **高一** |
| **2031** | **高二** |
| **2032** | **高三** |

> 锚点：2026 年 6 月入库 → Y=2025 → **小学五年级**；2026 年 9 月及以后入库 → Y=2026 → **预备班**。

## 录入（一句话）

母亲说「**伶俐** 默写错了」→ 更新词表：

- **新词**：追加一行，`状态=在练`，`连续对=0`；`入库日期`=当天；`年级`按上表自动填
- **已有词**：`状态=在练`，`连续对=0`，备注可补

**不问** Source、标签、错因、掌握度星级；**不走** §3.3 错题两步归档。

## 默写卷（`dictation_*`，与 `exam_*` 分离）

| 项 | 规则 |
|:---|:---|
| 路径 | 新卷 `02_自定义组卷/未完成/dictation_[YYMMDD]_[序号].{md,pdf}`；完成后 md+pdf 移至 `已完成/` |
| 稳定 stem | `dictation_[YYMMDD]_[序号]`（basename 无【】前缀） |

| 卷面 | `根据拼音写出汉字：{拼音}` + 横线；**禁止**印汉字、无原题图、**无变式** |
| PDF | `python3 scripts/md_to_exam_pdf.py "02_自定义组卷/未完成/dictation_….md"` |

## 批改（只改词表）

按题号反馈，如 `1对 2错` 或「伶俐对了」：

| 结果 | 词表操作 |
|:---|:---|
| **对** | `连续对 +1`；若 **≥2** → `状态=已掌握`，`连续对=0` |
| **错** | `状态=在练`，`连续对=0` |

整卷做完后：更新 `完成日期`；`dictation_*` md+pdf 从 `未完成/` 移至 `已完成/`（同 `exam_*` §5.2）。

权威规则：工作空间根目录 `.cursorrules` **§3.5**。

# 语文默写清单

> **与 `01_错题库` 分工**
>
> | 库 | 记什么 |
> |:---|:---|
> | **01_错题库** | 有题干、需复盘的阅读/作文/病句等 |
> | **本目录** | **字/词/成语**默写写错——只维护 [`默写词表.md`](./默写词表.md) 一张表 |

**禁止**为默写词条创建 `chn_*.md` 错题档案。

**组卷**：新默写走 **`memo_*`**（可与英语类型混卷）；`组默写` 或 `组默写 语文字形+短语`。旧 `dictation_*` 仅 legacy，不 retrofit。

## 数据源

唯一文件：**[`默写词表.md`](./默写词表.md)**

| 列 | 说明 |
|:---|:---|
| **词语** | 正确汉字 |
| **拼音** | 带声调，组卷时**只印此项** |
| **状态** | `在练` \| `已掌握` |
| **复习次数** | 历次已完成默写卷批改累计（不论对错） |
| **入库日期** | `YYYY-MM-DD` |
| **年级** | 由入库日期**自动填写** |
| **备注** | 可选 |
| **review_level** | `0–5`（艾宾浩斯档位） |
| **next_review** | 下次应复习日 |

`连续对` 为历史列，新批改不再写。复习规则见根目录 **§3.5.1**。

## 年级自动换算（与英语专名词表共用）

**学年切换日**：每年 **9 月 1 日**起算新学年。

给定**入库日期** `d`：月份 **≥ 9** → `Y =` 年份；否则 `Y =` 年份 **− 1**。

| 学年起始年 Y | 年级 |
|:---:|:---|
| ≤2024 | 小学四年级及以下 |
| **2025** | **小学五年级** |
| **2026** | **预备班** |
| **2027** | **初一** … |

## 录入（一句话）

母亲说「**伶俐** 默写错了」→ 更新词表：

- **新词**：`在练`，`review_level=0`，`next_review=入库日+1`；`入库日期`=当天
- **再错**：`在练`，`review_level=0`，`next_review=当天+1`

## 默写卷（`memo_*`）

| 项 | 规则 |
|:---|:---|
| 路径 | `02_自定义组卷/未完成/memo_[YYMMDD]_[序号].{md,pdf}` |
| 抽题 | 默认与其他类型合并组卷；单类 `组默写 语文字形` 每类最多 8 条（艾宾浩斯加权） |
| 卷面 | `根据拼音写出汉字：{拼音}` + 横线 |
| PDF | `python3 scripts/md_to_exam_pdf.py "02_自定义组卷/未完成/memo_….md"` |

## 批改（只改词表）

`批改 memo_260610_01：1对2错` 或按词条口头反馈：

| 结果 | 词表操作 |
|:---|:---|
| **对** | `复习次数+1`；`review_level+1`；更新 `next_review`；`≥2` → `已掌握` |
| **错** | `复习次数+1`；`在练`；`review_level=0`；`next_review=当天+1` |

权威规则：工作空间根目录 `.cursorrules` **§3.5.1**。

## 关联维护

凡错题写入或继承 `知识专题`，或批改新建 `#变式题` 档，**必须**同步专题 `关联错题` / `关联变式`、`## 关联错题` 表、子类 README。

**已掌握后新增关联档** → 专题**打回在练**；不批量改动其他已毕业错题。

---

## 毕业与打回

**仅操作者判断**；批改后**不**自动提示毕业。

### 毕业

口令：`… 已掌握` / `毕业 …`

有关联错题尚未毕业 → **先列出反问**；回 **`全部毕业`** 后落盘。均已毕业则直接落盘。

### 打回

已掌握专题因新增关联错题 → `status: 在练`，移回在练目录。

---

## 操作口令

| 口令 | 动作 |
|:---|:---|
| `建知识专题 …` | 新建专题 + 双向关联 |
| `同步 … 专题` | 更新规则精讲 |
| `组 … 专题` | 专题组卷（仅 `在练`） |
| `… 已掌握` / `毕业 …` | 毕业（有未毕业关联错题时先反问） |
| `全部毕业` | 确认后一并毕业 |

权威规则：仓库根目录 `.cursorrules` **§3.7**。

# 萌萌错题记录系统 (Sissy's Mistake Management System)

> [!IMPORTANT]
> 本系统采用标准化、数字化管理方式录入小升初错题。任何 AI 助理在操作前必须阅读本说明。

## 🛠️ 核心操作协议 (SOP)
本系统强制执行 **“两步走录入法”**，严禁未经确认直接生成档案：
1.  **第一步 (Diagnosis)**：AI 预判信息（科目、模块、考点）并提炼一句**【题目概括 (Title)】**，发送确认清单。
2.  **第二步 (Archive)**：用户确认概括无误并补充代码（如 A, M2, 3星）后，AI 执行**物理归位**（进入对应模块子目录）并生成档案。

## 📁 目录指南
- **[05_不熟练知识专区/](./05_不熟练知识专区/)**: **薄弱知识点专题**（规则精讲 + `组卷要求`）；首期 `英语/语法/`；原子题用 `知识专题` 链接；见该目录 `README.md` 与根目录 `.cursorrules` §3.7。
- **[03_英语默写/](./03_英语默写/)**: **专名 + 短语 + 句子 + 词性转换默写**（四张词表，不算错题）；见该目录 `README.md`。
- **[04_语文默写清单/](./04_语文默写清单/)**: **字词默写写错**（只维护 `默写词表.md`，不出 `chn_` 错题档）；默写卷 `dictation_*` 见该目录 `README.md`。
- **`scripts/pick_exam_weighted.py`**: 按错题 YAML **`错误次数`** 加权抽题（组卷在练槽）。
- **[.cursorrules](./.cursorrules)**: 错题中台**权威规则**（录入、组卷 §5、批改）；与 workspace 根目录 `.cursorrules` 联动。
- **[02_自定义组卷/组卷打印说明.md](./02_自定义组卷/组卷打印说明.md)**: 组卷**操作手册**（PDF 命令、自检）；规则以 `.cursorrules` 为准。
- **[02_自定义组卷/_exam_template.html](./02_自定义组卷/_exam_template.html)**: 组卷 HTML 排版模板；SVG 片段见 `_snippets/`。
- **[错题整理架构.md](./错题整理架构.md)**: 知识模块代码 (K1-K5)、错因代码 (M/C/E)、标签定义及 **AI 标签推荐规则**。
- **[00_复习全局看板.md](./00_复习全局看板.md)**: 学习状态监控中心。
- **02_自定义组卷/**: 组卷根目录（模板/手册）；卷子在 **`未完成/`** 或 **`已完成/`**，仅 **md + pdf**（如 `exam_260602_01`）。

## ✨ 复习卷高级特性 (Premium Features)
- **举一反三**：每道原题附变式；英语答题卡卷用 **`1.1、`** 印变式（不用【变式挑战】标题）。
- **数学卷**（`answer_sheet_mode: math`）：每题 **加粗题号 + 题干 + 题下虚线答题区**（`q-item` / `q-calc`）；**无**独立答题卡。详见 [`组卷打印说明.md`](./02_自定义组卷/组卷打印说明.md)。
- **英语答题卡三件套**（`answer_sheet_mode: english`）：**题卷 | 答题卡 | 答案** 合一 PDF；题卷 **`1、题干` 同行**、无考点标题；答案只填答题卡。详见 [`组卷打印说明.md`](./02_自定义组卷/组卷打印说明.md) · 根目录 `.cursorrules` §5.2.1。
- **视觉优化**：几何题保留截图；纯文字 OCR 排版。
- **数学/语文旧式卷**：题下仍可留答题区；英语新卷不在题卷留白。
- **一键 PDF**：`scripts/md_to_exam_pdf.py` 生成 A4 PDF（答题卡节自动分页）。

## 🏷️ 文件命名协议 (Naming Convention)
系统强制采用：`[科目]_[日期]_[序号]_[考点关键词].md`
> 示例：`math_20260515_01_圆柱表面积.md`


## 🤖 对 AI 助理的指令
如果你是第一次进入此目录的 AI，请立即阅读并学习 `.cursorrules` 中的规则，确保输出的 YAML 元数据和文件命名符合规范。

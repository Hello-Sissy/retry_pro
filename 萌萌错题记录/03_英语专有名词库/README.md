## 年级自动换算（与语文默写词表共用）

**学年切换日**：每年 **9 月 1 日**起算新学年。

给定**入库日期**日期 `d`：月份 **≥ 9** → `Y =` 年份；否则 `Y =` 年份 **− 1**。查表写入 **年级**：

| 学年起始年 Y | 年级 |
|:---:|:---|
| ≤2024 | 小学四年级及以下（2024→小四，2023→小三…） |
| **2025** | **小学五年级** |
| **2026** | **预备班** |
| **2027** | **初一** |
| **2028** | **初二** |
| **2029** | **初三** |
| **2030** | **高一** … |

## 录入

母亲说「**不认识 XX**」→ 更新词表：

- **新词**：追加一行，`状态=在练`，`连续对=0`，`入库日期`=当天，`年级`自动填
- **又忘了**：`状态=在练`，`连续对=0`

## 复习反馈

母亲说「**Switzerland 认对了**」/「又忘了」：

| 结果 | 词表操作 |
|:---|:---|
| **对** | `连续对 +1`；若 **≥2** → `状态=已掌握`，`连续对=0` |
| **错/忘** | `状态=在练`，`连续对=0` |

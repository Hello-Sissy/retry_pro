# 错题管理标准架构 (中台版)

> [!IMPORTANT]
> 本文档定义了“萌萌错题记录”系统的标准化字典。

## 1. 知识体系字典 (Modules)

### 数学 (Mathematics)
- **K1.数与代数**：计算、代数式、方程
- **K2.空间与几何**：图形认识、面积、体积
- **K3.典型应用题**：行程、工程、比例、盈亏
- **K4.数论基础**：质数、合数、因数、倍数
- **K5.统计与概率**：图表分析、概率计算

### 语文 (Chinese)
- **C1.基础知识**：字音字形、近义词、病句
- **C2.古诗文阅读**：背诵默写、诗歌鉴赏、文言文
- **C3.现代文阅读**：散文、说明文、议论文
- **C4.写作与表达**：作文、口语交际

### 英语 (English)
- **E1.词汇与语音**：单词拼写、发音
- **E2.语法体系**：时态、语态、词性切换
- **E3.阅读与完形**：信息提取、语境推断
- **E4.写作与翻译**：短文写作、句子翻译

## 2. 错因诊断代码 (Error Codes)

> 录入时由 AI **自动预判**；母亲回复 **「按推荐」** 即可。完整判定树见根目录 `.cursorrules` **[错因自动预判]**。

| 科目 | 代码 | 错因（归档描述） | 说明 |
| :--- | :--- | :--- | :--- |
| **数学** | M1 | **不会** | 概念/建模空白；§2 必填 **概念补洞** |
| | M2 | **思路卡壳** | 有尝试但模型或推理断档 |
| | M3 | **计算错误** | 方法对、算错/抄错/进率单位 |
| | M4 | **审题错误** | 看错问法、漏条件、范围误用 |
| **语文** | C1 | **不会** | 字词/常识不会 |
| | C2 | **审题** | 阅读偏差、未踩要点 |
| | C3 | **表达** | 写不出、表述弱 |
| **英语** | E1 | **不会** | 词义不明（专名→专有名词库） |
| | E2 | **语法** | 时态、搭配、句式 |
| | E3 | **理解** | 语境推断失败 |
| | E4 | **拼写** | 拼写、词形变化、大小写 |

**旧数学码映射（历史档只读）：** M1认知盲区→不会；M2逻辑断层→思路卡壳；M3→计算错误；M4习惯陷阱→审题错误；M5→思路卡壳。

### 2.2 英语默写（`03_英语默写/`，非错题）

与 `01_错题库` 分离：**专名没记住**、**短语/句子/词性转换默写写错**各记一张词表，用于复习；**不算错题**。

| 文件 | 记什么 |
|:---|:---|
| [`专有名词词表.md`](./03_英语默写/专有名词词表.md) | 国名/人名/地名/节气名/机构专名等 |
| [`短语词表.md`](./03_英语默写/短语词表.md) | 中译英短语默写 |
| [`句子词表.md`](./03_英语默写/句子词表.md) | 中译英整句默写 |
| [`词性转换词表.md`](./03_英语默写/词性转换词表.md) | 原词 → 目标词性转换默写 |

共用列：状态 / 连续对（满 **2** → `已掌握`）/ 入库日期 / 年级 / 备注。年级可按教材单元手填（如六年级上）。

详见 [03_英语默写/README.md](./03_英语默写/README.md) 与根目录 `.cursorrules` §3.6。

### 2.3 语文默写清单（`04_语文默写清单/`，非错题）

字形默写/听写写错 → **只维护** [`默写词表.md`](./04_语文默写清单/默写词表.md) 一张表；**禁止** `chn_*.md`。

| 列 | 说明 |
|:---|:---|
| 词语 / 拼音 | 正确字形与带声调拼音 |
| 状态 | `在练` \| `已掌握`（**连续写对 2 次** → 已掌握） |
| 连续对 | 写错清零；写对累加 |
| 入库日期 / 年级 | 见根目录 **[年级自动换算]** |
| 备注 | 可选 |

默写卷：`dictation_[YYMMDD]_[序号]`，卷面**仅拼音**。详见 [04_语文默写清单/README.md](./04_语文默写清单/README.md) 与根目录 `.cursorrules` §3.5。

### 2.4 不熟练知识专区（`05_不熟练知识专区/`，非错题）

与 `01_错题库` 分工：错题按**题**；专区按**知识点**合并深度规则。

| 路径 | 记什么 |
|:---|:---|
| `05_不熟练知识专区/英语/语法/` | 不熟练**语法**专题（首期） |
| 原子题 `知识专题` | basename 链接；**禁止 `组卷备注`** |
| 专题 `组卷要求` | 专项组卷指令**唯一出处** |

掌握度与错题同套 1–4。专题毕业**仅口令**（`… 已掌握`）；有未毕业关联错题则先反问，回 `全部毕业` 后移 `已掌握/` 并同步错题入 `已毕业/`。挂链 / 变式建档须同步 `关联错题` / `关联变式`；已掌握专题新增关联错题 → 专题打回在练。详见 [05_不熟练知识专区/README.md](./05_不熟练知识专区/README.md) 与根目录 `.cursorrules` §3.7。

### 2.1 错题 YAML：`错误次数`（组卷权重）

| 字段 | 说明 |
|:---|:---|
| `错误次数` | 整数，**默认 1**；组卷**主题再错**时 **+1**，且 **`mastery` 强制为 1**（只改 frontmatter，**不改** §2–§4） |
| 组卷抽取 | 在练槽按权重 `max(1, 错误次数)` 加权，**次数越高越易入选** |
| `母题` | 仅 `#变式题` 档案：指向母题 `id` |
| `qtype` | **仅英语必填**：`单选题` \| `填空题` \| `阅读填空` \| `完形填空` \| `多选题`；§1 须写卷面题型 |

**变式判错**：自动新建独立 md（`#变式题`），`错误次数: 1`；不覆盖母题正文。

### 2.2 错题 Markdown 正文结构（归档必填）

| 章节 | 内容 |
| :--- | :--- |
| §1 原题 | 图片 + OCR 文本 |
| §2 错因分析 | `[代码] 描述` + 轨迹拆解；**若 `[M1] 不会`**（数学/语文/英语对应科目 M1/C1/E1），必须含 `### 概念补洞（[M1] 必填）` |
| §3 正确解析 (SOP) | 题眼 + MVP 步骤 |
| **§4 本质分析**（全科目） | **`一句话快速概括`**（`>` 1 句）+ **`展开分析`**（2–4 条）；提炼考什么、怎么想，供口头带练 |

## 3. 题目分级标签 (Tags)

### 3.1 两层结构

| 层 | 标签 | 适用范围 |
|:---|:---|:---|
| **难度/题型** | `#压轴题` `#易错坑点` `#经典模型` `#基础送分` `#启发点` | 全科目；归档默认 **≤2 个** |
| **考纲** | **`#超纲`** | **仅数学**；与难度标签**叠加**；**不占**上述 2 个名额 |

> 旧 `#超纲拓展` → 历史档只读；新录入统一 `#超纲`。

### 3.2 标签定义（难度类）
- `#压轴题`：逻辑复杂，分值高
- `#易错坑点`：含陷阱或特殊限制
- `#经典模型`：需背诵模板/套路
- `#基础送分`：不该错的简单题
- `#启发点`：标注关键转折点，适用于“一点就透”的题目

### 3.3 `#超纲`（数学 · 考纲维）

- **打标**：概念/技巧超出**对口公办小五课内**；或 LSFD/民办预备卷中的**自定义名题**（如兰生数）。
- **不打标**：预备班月考常规压轴（互素、短除 lcm、M/7 循环节等）→ 用 `#压轴题` / `#经典模型` 即可。
- **YAML 示例**：
  ```yaml
  tags:
    - '#压轴题'
    - '#经典模型'
    - '#超纲'
  ```
- **组卷**：默认复习卷**不抽** `#超纲`；需练再说「含超纲」。

### 3.4 录入时的 AI 标签推荐（强制流程）
错题录入第一步，AI **必须先输出标签推荐**，母亲确认后再写入 YAML；不可只列 A–F 选项让母亲从零选。

| 字段 | 规则 |
| :--- | :--- |
| **主推** | 必填 1 个，附一句理由 |
| **备选** | 0–1 个，仅当题型明显复合时给出 |
| **上限** | 难度标签默认 ≤2；数学 `#超纲` 可额外 +1 |
| **确认** | 母亲可回复「按推荐」、改字母/标签名，或「按推荐，加超纲」 |

**推荐触发速查（难度类）**

| 标签 | 何时主推 |
| :--- | :--- |
| `#压轴题` | 多步综合、末题难度、思维链≥3 步 |
| `#易错坑点` | 「任意/都/一定」、边界反例、概念陷阱、判断易错 |
| `#经典模型` | 牛吃草、相遇追及、割补、工程等有固定模型 |
| `#基础送分` | 单知识点、≤2 步、本应满分 |
| `#启发点` | 一个题眼打通全题，适合复盘「顿悟点」 |
| **`#超纲`** | 超出小五课内 / LSFD 自定义拓展（**数学**） |

## 3. 掌握度分级定义 (Mastery Levels)

| 等级 | 状态名称 | 具象化描述 (萌萌视角) | 复习建议 |
| :--- | :--- | :--- | :--- |
| ⭐ | **全盲/初识** | 题目读不懂，看答案解析也觉得像“天书”。 | 建议推倒重来，重新听知识点讲解。 |
| ⭐⭐ | **模糊/吃力** | 看解析能懂，但合上书自己完全写不出过程。 | 需要多次对照解析进行模仿练习。 |
| ⭐⭐⭐ | **观察/夹生** | 能做对，但中间需要翻书、看提示，或者耗时极长。 | 重点观察，3天后必须二次重做。 |
| ⭐⭐⭐⭐ | **熟练/巩固** | 能独立、快速做对，但细节（单位、符号）偶尔出错。 | 进入常规复习池，定期抽测。 |
| ⭐⭐⭐⭐⭐ | **通关/精通** | 看到题秒出思路，不仅做对，还能给家长讲明白。 | 正式通关，移出高频复习池。 |

> [!NOTE]
> **#压轴题 毕业规则（v3.0）**：组卷首次判对 → **4 星** + `压轴题二度验证: 一度通过`；**连续第二次**组卷判对（中间无判错）→ **5 星**并移入 `已毕业/`。详见工作空间 `.cursorrules` §5.3。
>
> **Sprint Review Slot**：默认组卷 **8** 道在练 + **2** 道 `已毕业/` 复检；题卷标 **`〔复检〕`**；复检题做错 → **`错误次数`+1**、**`mastery`→1**、移回在练（见 `.cursorrules` §5.3，已废止 A/B/C/D 矩阵）。

# 英语默写

> **与 `01_错题库` 分工**
>
> | 库 | 记什么 |
> |:---|:---|
> | **01_错题库** | 语法/思路/拼写规则错了 |
> | **本目录** | **专名没记住**、**短语/句子/词性转换默写写错**——只维护词表，用来复习；**不算错题** |

读题只是「不认识 Switzerland」→ **专有名词词表**加一行。  
短语默写错了 → **短语词表**；整句翻译默写错了 → **句子词表**；词性转换错了 → **词性转换词表。

**组卷**：新默写一律走 **`memo_*`**（§3.5.1 艾宾浩斯）；`组默写` 或 `组默写 短语+词性转换`。

## 年级自动换算（与语文默写词表共用）

**学年切换日**：每年 **9 月 1 日**起算新学年。

给定**入库日期** `d`：月份 **≥ 9** → `Y =` 年份；否则 `Y =` 年份 **− 1**。查表写入 **年级**：

| 学年起始年 Y | 年级 |
|:---:|:---|
| ≤2024 | 小学四年级及以下 |
| **2025** | **小学五年级** |
| **2026** | **预备班** |
| **2027** | **初一** |
| **2028** | **初二** |
| **2029** | **初三** |
| **2030** | **高一** … |

## 复习反馈（艾宾浩斯）

| 结果 | 词表操作 |
|:---|:---|
| **对** | `复习次数+1`；`review_level+1`；`next_review=批改日+下一档`；`review_level≥2` → `已掌握` |
| **错/忘** | `复习次数+1`；`在练`，`review_level=0`，`next_review=批改日+1` |

间隔（天）：`1 → 3 → 7 → 14 → 30 → 60`。**已掌握**条目到期仍会进 `memo_*` 卷。

权威规则：工作空间根目录 `.cursorrules` **§3.5.1**、**§3.6**。

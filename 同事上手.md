# 同事上手 · MistakeMaster 错题中台

> 仓库：<https://gitee.com/sissy-chen/MistakeMaster>  
> 这是一套可 fork 的**模板**：规则 + 目录结构 + 少量样例题，不含家庭隐私数据。

---

## 1. 你需要准备什么

| 项 | 要求 |
|:---|:---|
| **Cursor**（推荐）或能加载 `.cursorrules` 的 IDE | 打开仓库**根目录**（含 `.cursorrules` 的那一层） |
| **Python 3** | 组卷后生成 PDF |
| **Git** | `git clone` 本仓库 |

克隆：

```bash
git clone https://gitee.com/sissy-chen/MistakeMaster.git
cd MistakeMaster
```

用 Cursor：**文件 → 打开文件夹** → 选 `MistakeMaster`（不要只打开里面的 `萌萌错题记录`）。

---

## 2. 五分钟跑通一条链

在 Cursor 对话框依次试（样例数据已内置）：

| 步骤 | 你对 AI 说 | 预期 |
|:---|:---|:---|
| ① 看样例 | 「打开 `eng_20260601_05_advice不可数.md` 讲结构」 | 讲解 YAML + 四段正文 |
| ② 组卷 | **`组英语`** 或 **`组`** | 先出确认清单，你回 **`组`** 后落盘 `exam_*.md` |
| ③ 出 PDF | （AI 应自动执行；也可手动）见下 | 同目录生成 `.pdf` |
| ④ 批改 | **`批改 exam_260603_01：1对，2错`**（按卷面改题号） | 更新错题 YAML 的 `mastery` / `错误次数` |

手动出 PDF（在 `萌萌错题记录` 目录下）：

```bash
cd 萌萌错题记录
python3 scripts/md_to_exam_pdf.py "02_自定义组卷/已完成/exam_260603_01.md"
```

打印：**英语新卷**只给孩子「第一部分 + 答题卡」；**第三部分标准答案**和 **深度分析** 勿发孩子（可撕页）。

## 3. 目录是干什么的

```
MistakeMaster/
├── .cursorrules              ← AI 总规程（必加载）
├── 同事上手.md               ← 本页
└── 萌萌错题记录/
    ├── 01_错题库/            ← 每题一个 .md（在练 / 已毕业）
    ├── 02_自定义组卷/        ← exam_*.md；未完成/ → 已完成/
    ├── 03_英语专有名词库/    ← 专名表（不算错题）
    ├── 04_语文默写清单/      ← 默写词表（不算错题）
    ├── assets/images/        ← 原题截图
    └── scripts/              ← 加权抽题、md→PDF
```

**样例题（可删可改）**

| 文件 | 用途 |
|:---|:---|
| `01_错题库/英语/.../eng_20260601_05_advice不可数.md` | 在练 · 语法 · 含概念要点 |
| `01_错题库/英语/.../已毕业/eng_20260601_01_sweets...md` | 已毕业样例 |
| `01_错题库/数学/.../math_20260531_01_互素...md` | 数学 · 含概念补洞 |
| `01_错题库/数学/.../math_20260514_03_牛吃草...md` | 经典模型样例 |
| `02_自定义组卷/已完成/exam_260603_01.md` | 英语答题卡卷样例 |

## 4. 日常口令（记这 4 个就够）

| 场景 | 口令 |
|:---|:---|
| 录入确认 | **`按推荐`** |
| 只改错因 | `8题：审题` |
| 组卷 | **`组`** / **`组数学`** / **`组英语`** |
| 批改 | **`批改 exam_YYMMDD_nn：1对，2错，1.1对`** |

录入：发**错题截图** → AI 反问标签/错因 → 你回 `按推荐` 才写入 `01_错题库/`。

## 5. 自己孩子怎么用（定制）

1. **Fork** 本仓库到你的 Gitee/GitHub（建议**私有**）。
2. 改 `错题整理架构.md` 里的模块/年级口径（可选）。
3. 删掉 `01_错题库` 样例题，从自家试卷开始录入。
4. 根目录 `.cursorrules` 很长但可只改「考纲锚点」「默认题量」等少量句子；细则在 `萌萌错题记录/.cursorrules`。

**不要**把 `05_已做的组卷/`、`错题原题/` 大文件夹推进 Git（已在 `.gitignore` 声明）。

## 6. 常见问题

**Q：打开子文件夹 AI 不听话？**  
A：必须打开 **MistakeMaster 根目录**，让 Cursor 读到根 `.cursorrules`。

**Q：图片裂了？**  
A：错题 md 引用 `assets/images/`；录入新题时把原图放进该目录。

**Q：仓库里没有 PDF？**  
A： intentional。克隆后本地跑 `md_to_exam_pdf.py` 生成（见 §2）。

**Q：和「错题 App」比？**  
A：数据在 Markdown/Git 里，规则你可改；AI 负责 OCR、诊断、组卷、批改回写。

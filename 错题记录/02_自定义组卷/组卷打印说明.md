## 答题卡三件套（新卷 · 卷面结构）

指 **同一 md 内** 三节：题卷 | 答题卡 | 答案 → 合并为一个 PDF。

| 部分 | 打印给谁 | 批改时 |
|:---|:---|:---|
| **第一部分 · 题目** | 学生审题、草稿 | 不上传 |
| **第二部分 · 答题卡** | 学生**只在这里写最终答案** | **上传本节** 或你打字 `1对，1.1错` |
| **第三部分 · 标准答案** | 仅家长 | **不要**传给 AI（已在 md 的 `answers`） |

- **英语**：`answer_sheet_mode: english`，题卷无作答区。  
- **英语阅读+概要**：题卷须**短文全文 + 概要全文**（空位写 `______`，PDF 脚本转可见横线）；**无** `N.1` 变式；答题卡变式列 **—**。  
- **数学**：`answer_sheet_mode: math`（新卷默认），题卷无作答区，**全部最终答案写在答题卡**；`mapping[n].qtype` 分配留白（见下表）。  
- **数学旧式**：`math_selective` 仍可用（选择/单空上卡，计算/应用在题卷留白）。  
- **复检题**：题号连续；题卷 `N、〔复检〕` 连写；答题卡行标 `〔复检〕`。  
- **旧卷**（无 `answer_sheet_mode`）：数学/语文新卷须附 **`## 标准答案`**；旧已完成卷不强制 retrofit。  
- **`dictation_*`**：不适用三节结构。

## 数学/语文 legacy 卷（无 `answer_sheet_mode`）

- 卷末 **`## 标准答案`** 表（题号 | 主题 | 变式），与 YAML `answers` 同步。
- 打印时可整卷打印（含答案页），或自行撕掉答案页给孩子。
- `answers` 示例：`1: { main: "5/12", "1.1": "7/30不能", "1.2": "11/40能" }`（数学双变式用 `"N.1"`/`"N.2"`；单变式仍可用 legacy 键 `variant` 兼容旧卷）

## 生成 PDF

```bash
cd "错题记录"
python3 scripts/md_to_exam_pdf.py "02_自定义组卷/未完成/exam_YYMMDD_nn.md"
```

- 输出：同目录 `exam_YYMMDD_nn.pdf`  
- **页脚**：每页居中 **「第 n 页 / 共 m 页」**（`md_to_exam_pdf.py` 的 `@page` 页脚；`exam_*` / `dictation_*` 通用）  
- 第二节/第三节/`标准答案` 节前自动分页；旧卷顶状态徽章自动剔除  
- 图片路径在 md 中用 `../../assets/images/…`

脚本失败且重试仍失败时，将**完整终端输出**反馈给 Agent 排查；勿在卷子目录手写 `.html`。

## 卷面通用

- **全科目不写**「建议用时」；**禁止**卷顶完成状态徽章及 H1 状态前缀。
- **数学 / 英语 · 概念热身**：本卷有「不会」题时 H1 下插入 `## 概念热身`（**不写**引导 blockquote；只保留 `### 概念名` + bullet）。不算题号、不进 `answers`。
- **深度分析（家长区）**：错题库 **`错误次数` ≥ 2** 的题，组卷**必须**在 `## 第三部分 · 标准答案` 之后写 `## 深度分析`（再写「附：题库映射」）。**勿发孩子**；改排版时不得删本节。详则见根目录 **[深度分析]**。
- **第一部分题型子标题**：`### **一、单选题**` 等（小于 `## 第一部分`）；`qtype` 用单选题/多选题等，不写「选择题」。
- **数学/语文 legacy 卷**：须 YAML `answers` + 卷末 `## 标准答案`。
- **数学分数 / 公式**：组卷 md 可沿用错题库 LaTeX；`md_to_exam_pdf.py` 自动转 HTML 分数。改 md 后须重跑脚本。

### 批改示例

```text
批改 exam_260602_03：1对，2错，1.1对，9错
```

或上传答题卡照片 + `批改 exam_260602_03`。存疑：`3改对`。

AI 按 stem 在 `未完成/`、`已完成/` 查找 `{stem}.md`。

## 题型标记 `qtype`（组卷 YAML · 仅分配答题卡空间）

卷面**不印**题型名称；组卷时按 `qtype` **排序**（同类型相邻），再在答题卡按题型留白。

| 科目 | 可选 `qtype` | 答题卡留白 |
|:---|:---|:---|
| **数学** | 选择题 / 判断题 / 填空题 / 计算题 / 应用题 | 选择·判断 → `blank-short`×1（不印选项）；填空 → `blank-line`×空数；计算 → 仅最终结果×1，**须写过程**（如短除法）×2~3；应用 → 单数字结果×1 |
| **英语** | 单选题 / 多选题 / 填空题 / 阅读填空 / 完形填空 | 单选/多选 `blank-short`（**不印** ABCD）；填空/阅读 `blank-line` |

**横线数量**：按**每道小问要写几个答案**定，不按 `qtype` 机械套多行。YAML 可加 `sheet_lines`（主题行）/`process: true`（须写算式过程）。

## 英语卷（`answer_sheet_mode: english`）

- **题卷**：`1、题干…` **同一行**；**不写**考点名、ENG-ID；H1 下**无**说明 blockquote（见卷末映射表）。  
- **答题卡**：`table.sheet` 仅两列 **题号 | 答案**；主题与 1.1/1.2 **同一行**并排（`sheet-compact` + `q-sub` 标号）；单选 `blank-short`；填空 `blank-mid`；**不印**「主题/变式」分列，**不**把 1.1 单独占一行。  
- 排版片段参考 `_exam_template.html`。

## 自检

1. 打开 `.pdf`，确认三节、答题卡表、答案与 YAML `answers` 一致  
2. 打印：前两节给孩子，第三节勿夹入  
3. 改 md 后须重跑 `md_to_exam_pdf.py` 再打印

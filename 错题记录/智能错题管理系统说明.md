# 智能错题管理系统 (Smart Mistake-Management System)

> [!IMPORTANT]
> 这是一套专为学生“小升初”设计的闭环式错题管理方案。它不仅是题目的堆砌，更是通过数据驱动、AI 辅助，实现从“发现错题”到“彻底掌握”的智能化中台。

---

## 🚀 核心功能模块

### 1. 结构化错题库 (`01_错题库`)
- **原子化存储**：按学科（语/数/外）和知识体系（K1-K5 等）精细化分类。
- **深度诊断**：每道题都包含 `Mastery` (掌握度) 和 `Error Code` (错因诊断)，精准定位学生的薄弱环节（如：M2 逻辑断层）。
- **举一反三**：每道错题自动关联变式题思路，防止机械记忆。

### 2. 学情追踪看板 (`00_复习全局看板.md`)
- **动态可视化**：通过“攻坚、稳固、通关”三个维度实时展示全科掌握现状。
- **优先级管理**：自动筛选 1-2 星题目进入“紧急攻坚”区，确保复习精力用在刀刃上。

### 3. 自动组卷引擎 (`scripts/gen_exam.py`)
- **一键成卷**：根据掌握度和标签（如“#压轴题”）自动抽题。
- **出版级 PDF**：自动调用 Chrome 内核，生成排版精美、带答题留白、带矢量几何图的 PDF 试卷。
- **矢量绘图**：内置 SVG 精准绘图功能，确保几何题目在打印时依然清晰。

---

## 🛠️ 标准工作流 (Workflow)

1.  **录入 (Input)**：上传错题截图，AI 进行 OCR、考点诊断，并**主动推荐标签**（主推 + 可选备选，附理由）；母亲确认或修改后，再补充来源、错因、掌握度。
2.  **归档 (Archive)**：AI 根据确认后的元数据，将文件物理归位到对应的知识子目录。
3.  **追踪 (Track)**：看板实时更新，标记复习时间点（如：夹生题 3 天后重做）。
4.  **复习 (Review)**：运行组卷脚本，生成 PDF 练习册。
5.  **回写 (Feedback)**：根据重做结果更新掌握度，直至题目进入“完美通关”池。
